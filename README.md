# MIS 432: AI in Business
### A free, case-based textbook teaching machine learning strategy through real companies.
 
> **Live site:** https://prof-califf.github.io/ai-in-business
 
---
 
## What This Is
 
A free, open web textbook built for upper-division business students with no prior coding experience. Every chapter starts with a real company facing a real problem, uses that story to introduce core machine learning concepts, and ends with a hands-on lab where students build a simplified version of the system in Python using Claude and Google Colab.
 
No textbook to buy. No software to install. Just a link.

## The Thread That Connects Every Chapter
 
Every company in this textbook follows the same underlying pattern — the **AI Factory**:
 
```
Data → Model → Prediction → Decision → Value → (back to Data)
```
 
EveryCure uses it to find drug treatments. Netflix uses it to test artwork. Spotify uses it to personalize music. Learning to see this pattern across industries is the core skill of the course.
